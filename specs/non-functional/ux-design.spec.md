# UX Design Specification

The detailed guidance has been split into focused sub-specs under `specs/non-functional/ux/` to make it easier to work on one aspect of the design at a time.

**How to use this spec**
- Start here to find the relevant section.
- Follow the link for the full authoritative content.
- Avoid duplicating rules in feature specs; link to the relevant section instead.
